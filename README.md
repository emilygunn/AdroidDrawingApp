# Android Drawing App

A simple drawing app built using Android Studio and Kotlin.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Clone the repository:  
   git clone https://github.com/emilygunn/AndroidDrawingApp
2. Open in Android Studio
3. Run on:
    - Emulator OR
    - Physical Android device

Instructions for using the software:

1. Simply draw on the screen with your finger.
2. Various buttons will allow you to undo and pick colors.

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Android Studio (current)
